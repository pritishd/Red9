Pro_Pack : Exporter¶

ProPack exporter module is our own export management system for handling the output of files based on the metaData export node class systems.

>>> # import statement for the module via the r9pro decompiler
>>> from Red9.pro_pack import r9pro
>>> r9pro.r9import('r9exp')
>>> import r9exp
